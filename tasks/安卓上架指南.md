# 专注钟 - 安卓应用上架指南

## 项目概述

专注钟番茄钟应用已通过 **Capacitor** 封装为原生安卓应用，支持生成 APK / AAB 安装包并上架 Google Play 或其他安卓应用市场。

---

## 快速构建（一键打包）

### 环境要求

- **Node.js** 18+
- **JDK** 17+
- **Android Studio** 最新稳定版
- **Android SDK** (compileSdk 35, minSdk 23, targetSdk 35)

### 构建步骤

#### 1. 构建 Web 应用

```bash
cd /workspace/app-bmqdxv66ipkx
npx vite build
```

#### 2. 同步到安卓平台

```bash
npx cap sync android
```

#### 3. 打开 Android Studio 构建

```bash
npx cap open android
```

在 Android Studio 中选择 **Build → Generate Signed Bundle / APK**，选择 APK 或 AAB 格式进行构建。

#### 4. 命令行构建（可选）

```bash
cd android
./gradlew assembleRelease
```

构建完成后，APK 位于：
```
android/app/build/outputs/apk/release/app-release.apk
```

- **密钥库文件**：`android/app/release-key.keystore`
- **密钥库密码**：`pomodoro123`
- **密钥别名**：`pomodoro-key`
- **密钥密码**：`pomodoro123`
- **有效期**：27年（10,000天）

### 更换正式签名密钥

如需替换为正式上架密钥：

1. 生成新密钥：
```bash
keytool -genkey -v -keystore my-release-key.keystore -alias my-alias -keyalg RSA -keysize 2048 -validity 10000
```

2. 更新 `android/app/build.gradle` 中的 signingConfigs：
```gradle
signingConfigs {
    release {
        storeFile file("my-release-key.keystore")
        storePassword "YOUR_STORE_PASSWORD"
        keyAlias "my-alias"
        keyPassword "YOUR_KEY_PASSWORD"
    }
}
```

## 上架到 Google Play

### 准备工作

1. 注册 [Google Play 开发者账号](https://play.google.com/console)（一次性费用 $25）
2. 准备应用截图（手机 + 平板各 2-3 张）
3. 准备应用图标（512x512 PNG）
4. 准备应用描述、隐私政策 URL

### 上架流程

1. 登录 [Google Play Console](https://play.google.com/console)
2. 创建新应用，填写应用名称和语言
3. 进入 **版本管理 → 生产版本 → 创建新版本**
4. 上传 AAB 文件（推荐）或 APK 文件
5. 填写商店列表信息：
   - 应用标题：专注钟 - 番茄钟时间管理
   - 简短描述：番茄钟时间管理工具，助你专注工作、高效完成任务
   - 完整描述：详细功能介绍...
   - 屏幕截图、应用图标、置顶大图
6. 设置内容分级问卷
7. 设置定价和发布范围
8. 提交审核（通常 1-3 个工作日）

## 常见问题

### Q: 构建失败，提示 Gradle 版本不匹配？
A: 在 Android Studio 中点击 **File → Sync Project with Gradle Files**，或运行 `./gradlew wrapper --gradle-version 8.7`

### Q: 应用图标没有更新？
A: 卸载旧应用后重新安装，或清理 Android Studio 缓存 **Build → Clean Project → Rebuild Project**

### Q: Web 资源没有同步？
A: 运行 `npx cap sync android` 确保 dist 目录内容已复制到安卓项目

### Q: 如何更新应用版本？
A: 修改 `android/app/build.gradle` 中的 `versionCode` 和 `versionName`，然后重新构建
