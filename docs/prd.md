# 需求文档

## 1. 应用概述

### 1.1 应用名称
番茄钟

### 1.2 应用描述
一个前端实现的番茄钟时间管理工具，帮助用户通过番茄工作法提高专注力和工作效率。用户可以设置工作时长、休息时长和长假时长，应用会进行倒计时并记录完成的番茄钟数量。界面简洁美观，所有数据存储在浏览器本地。

## 2. 用户与使用场景

### 2.1 目标用户
需要时间管理和专注力提升的用户，包括学生、职场人士、自由职业者等。

### 2.2 核心使用场景
- 用户需要专注工作或学习一段时间
- 用户希望通过番茄工作法管理时间
- 用户需要记录自己的工作成果

## 3. 页面结构与功能说明

### 3.1 页面结构
```
番茄钟应用
├── 主页面
│   ├── 倒计时显示区域
│   ├── 控制按钮区域
│   ├── 设置区域
│   └── 统计区域
```

### 3.2 主页面功能说明

#### 3.2.1 倒计时显示区域
- 显示当前倒计时剩余时间（格式：MM:SS）
- 显示当前状态（工作中/休息中/长假中）

#### 3.2.2 控制按钮区域
- 开始按钮：启动倒计时
- 暂停按钮：暂停或恢复倒计时（切换功能）
- 恢复按钮：重置当前倒计时到初始时长并回到初始状态
- 按钮状态规则：
  - 初始状态：开始按钮可用，暂停按钮和恢复按钮禁用（呈现灰色）
  - 点击开始后：开始按钮禁用，暂停按钮和恢复按钮变为可用状态
  - 点击暂停后：倒计时暂停，再次点击暂停则倒计时恢复
  - 点击恢复后：倒计时重置为当前状态的初始时长，开始按钮变为可用，暂停按钮和恢复按钮变为禁用

#### 3.2.3 设置区域
- 番茄钟时长设置：用户输入工作时长（单位：分钟）
- 休息时长设置：用户输入短休息时长（单位：分钟）
- 长假时长设置：用户输入长休息时长（单位：分钟）
- 保存设置按钮：保存用户设置的时长到浏览器本地存储

#### 3.2.4 统计区域
- 显示今日完成的番茄钟数量
- 显示总完成的番茄钟数量

#### 3.2.5 提醒音效
- 工作倒计时结束时播放提醒音效
- 休息倒计时结束时播放提醒音效

## 4. 业务规则与逻辑

### 4.1 倒计时规则
- 用户点击开始按钮后，根据当前状态开始倒计时
- 首次启动默认为工作状态，使用番茄钟时长
- 倒计时结束后自动切换状态：工作结束→休息开始，休息结束→工作开始
- 每完成4个番茄钟后，下一次休息使用长假时长，否则使用休息时长

### 4.2 数据存储规则
- 用户设置的时长保存在浏览器本地存储中
- 完成的番茄钟数量保存在浏览器本地存储中
- 今日完成数量在每日0点自动重置

### 4.3 默认时长
- 番茄钟时长：25分钟
- 休息时长：5分钟
- 长假时长：15分钟

### 4.4 UI设计原则
- 界面精致、现代，符合专注力工具特性
- 按钮状态变化清晰可见（可用/禁用状态区分明显）
- 整体视觉风格简洁、专注

## 5. 异常与边界情况

| 场景 | 处理方式 |
|------|----------|
| 用户输入非数字时长 | 提示用户输入有效数字 |
| 用户输入时长小于1分钟 | 提示用户时长不能小于1分钟 |
| 用户输入时长大于120分钟 | 提示用户时长不能超过120分钟 |
| 倒计时进行中关闭浏览器 | 下次打开时倒计时重置 |
| 浏览器不支持本地存储 | 使用默认时长，统计数据不保存 |

## 6. 验收标准

1. 用户打开应用，看到倒计时显示为25:00，状态显示为「工作中」，开始按钮可用，暂停按钮和恢复按钮禁用呈现灰色
2. 用户点击开始按钮，倒计时开始递减，开始按钮禁用，暂停按钮和恢复按钮变为可用状态
3. 用户点击暂停按钮，倒计时暂停，再次点击暂停按钮，倒计时恢复
4. 倒计时结束后，播放提醒音效，状态自动切换为「休息中」，倒计时显示为05:00
5. 用户点击恢复按钮，倒计时重置为当前状态的初始时长，开始按钮变为可用，暂停按钮和恢复按钮变为禁用
6. 用户完成一个完整的工作倒计时，统计区域显示今日完成1个番茄钟
